# 🚧 Road Anomaly Dataset

## 📌 Description
This dataset contains images of road anomalies used for AI-based detection:
- Potholes
- Cracks
- Speed Bumps
- Normal Roads

## 📂 Dataset Structure
potholes/
cracks/
bumps/
normal/
